如何用Github Pages+Hexo搭建个人博客

Posted on 2020-09-26 Edited on 2020-10-13 In 杂

前言

这是我的第一篇博客。第一次尝试用Github Pages+Hexo这种方式搭建自己的个人技术博客，那么第一篇博客就介绍下建站过程吧。感谢我的璐宝的全程见证。

依赖环境

  • NodeJs
  • Git

安装Hexo并初始化博客

浏览器访问localhost:4000即可看到默认的博客页面。

Github Pages搭建个人网站

浏览器访问http://username.github.io即可，这就是你托管在Github上的个人网站。

本地Hexo博客部署到Github上

如果出现Permission denied的问题大概率是你本地的ssh公钥没有配置到你的Github上。这里略过。

顺利的话，过一会儿再刷新http://username.github.io即可看到本地的博客页面出现。

结语

简单来讲就是这么一个过程。更多内容等我更熟悉了再补充。
